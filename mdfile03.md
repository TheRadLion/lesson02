# Bullet Points and Links

To make a list:

* Add a asterisk to the start of the line
* Then add a space
* Finally, add your text
It's *that* simple!

# Links

Basically you add these arrow thingys, <>, to the ends of the link.
Like so:

> <https://commonmark.org/help/tutorial/07-links.html>

That's *It*, **WaHoo!**
